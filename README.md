=======
codedryer
=========

Measure redundancy in your code base and DRY up your code. Many languages supported

## Installation

Add this line to your application's Gemfile:

    gem 'codedryer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install codedryer
